## 1\. Get your Portkey API Key

[Create](https://app.portkey.ai/signup) or [log in](https://app.portkey.ai/login) to your Portkey account. Grab your account's API key from the "Settings" page.

<Frame caption="Copy your Portkey account API key">
  <img src="/images/welcome/welcome-2.gif" alt="Copy your Portkey account API key" />
</Frame>

Based on your access level, you might see the relevant permissions on the API key modal - tick the ones you'd like, name your API key, and save it.

## 2\. Choose Your Integration Method

Portkey offers multiple integration options to suit your development environment:

#### a. Portkey SDKs

Install the Portkey SDK for your preferred language:

**Python:**
```bash
pip install portkey-ai
```

**JavaScript:**
```bash
npm install portkey-ai
```

#### b. REST APIs

Integrate directly using Portkey's REST API endpoints.

#### c. OpenAI SDK Compatibility

Use Portkey with any OpenAI SDK by changing the base URL.

## 3. Make Your First Request

Here are examples of how to make a request using different integration methods:

#### Using Portkey Python SDK

```python
#portkey python sdk
```

#### Using Portkey JavaScript SDK

```javascript
//portkey javascript sdk
```

#### Using REST API

```bash
#portkey cURL integration
```

#### Using OpenAI Python SDK with Portkey

```python
#portkey openai python sdk with portkey
```

#### Using OpenAI Node SDK with Portkey

```node
//portkey openai node sdk with portkey
```

#### Using OpenAI Java SDK with Portkey

```java
//portkey openai java sdk with portkey
```

#### Using OpenAI Go SDK with Portkey

```go
//portkey openai go sdk with portkey
```
