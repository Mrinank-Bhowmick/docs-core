**Portkey Provider Slug:** `novita-ai`

## Overview

Novita AI is an AI Cloud Platform that provides Model APIs. This document outlines the features, supported models, and integration methods available for Novita AI through the Portkey platform.

## Quick Links

- [Novita AI Website](https://novita.ai/)
- [Pricing](https://novita.ai/model-api/pricing)
- [Documentation](https://novita.ai/docs/get-started/quickstart.html)
- [Community](https://discord.gg/a3vd9r3uET)

## Supported Features

### Supported Models

| Type | Description |
|------|-------------|
| Chat Completions | Portkey supports all OpenAI compliant chat completions API for Novita AI models |

[View all supported models](https://novita.ai/model-api/product/llm-api)

### Unsupported Features

- **Prompt Playground**: Prompt playground is not supported for Novita AI models
- **Unsupported Endpoints**:
  - Image Generator
  - Image Editor
  - Video Generator
  - Audio
  - Face Editor

1. **Generate a Virtual Key**
   Get your API key from Novita AI and add it to Portkey to create a virtual key.

   You can get your Novita AI API key from the Novita AI website [here](https://novita.ai/docs/get-started/quickstart.html#_2-manage-api-key).
