**Portkey Provider Slug:** `nomic`

## Overview

Nomic provides embedding models for developers and enterprises to build AI-powered applications. This document outlines the features, supported models, and integration methods available for Nomic through the Portkey platform.

## Quick Links

- [Nomic Website](https://www.nomic.ai/)
- [Pricing](https://atlas.nomic.ai/#pricing)
- [Documentation](https://docs.nomic.ai/index.html)
- [Community](https://discord.com/invite/myY5YDR8z8)

## Supported Features

### Supported Models

| Type | Models |
|------|--------|
| Text Embedding | nomic-embed-text-v1, nomic-embed-text-v1.5 |

Portkey supports the text embedding API for Nomic models.

[More models](https://docs.nomic.ai/atlas/models/text-embedding)

### Unsupported Features

- **Prompt Playground**: Prompt playground is not supported for Nomic models
- **Image Embedding**: Portkey currently does not support the image embedding API for Nomic models (unsupported models: nomic-embed-vision-v1, nomic-embed-vision-v1.5)
